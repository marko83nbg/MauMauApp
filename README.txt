Tema:
Aplikacija za pracenje rezultata u kartaskoj igri Mau Mau

Opis: 
Aplikacija sluzi za pracenje rezultata i evidentiranje pobednika.
Nakon svakog deljenja tj. zavrsene partije moguce je dodati svakom igracu ostvaren rezultat.
Na kraju igre moguce je upisati pobednika.

Tehnologije:
HTML, CSS, JavaScript, Bootstrap biblioteka

Struktura projekta:
U root folderu nalaze se HTML fajlovi. Pocetna stranica je index.html. 
Fajl za stil nalazi se u folderu css. 
Fajl za JavaScript nalazi se u folderu js.
Fotografije se nalaze u images folderu
